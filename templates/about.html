About Page

Hi! Thanks for checking out my capstone project!

This site is meant to be used as support to a training program within a given sport. If I were a swimmer getting only swim workouts from my coach, I would use this app to generate a weight program to support my lifts and improve my performance in the pool.

Now for some of the technical stuff:

This site is written is created in a flask environment (python) with an SQLite database. The algorithm is "pseudo AI" constraint handling where the user gives constraints and the algorithm creates a workouts based around these parameters. In this case, sport and skill level. Overall this project was really eye opening to the front end side of programs rather than just the algorithms behind them.
